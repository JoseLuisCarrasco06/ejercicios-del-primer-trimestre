Ejemplo de párrafos y encabezados

Alineación a la izquierda

Este párrafo está alineado a la izquierda. Es la alineación predeterminada en la mayoría de los navegadores y es la más común para textos largos.

Alineación centrada

Este párrafo está centrado. Es útil para títulos, citas o para destacar textos en la página.

Alineación a la derecha

Este párrafo está alineado a la derecha. Se usa menos frecuentemente, pero puede ser útil para fechas o firmas en documentos.

Alineación justificada

Este párrafo está justificado, es decir, el texto se ajusta para que ambos márgenes queden alineados, generando un bloque uniforme. Se usa mucho en libros y revistas.
